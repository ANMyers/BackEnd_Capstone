About

This page is designed to help you understand more about the algorithms used on this site and try them out for yourself without needed any mathematical background or comfort in writing code.

Currently the site is using Nearest Neighbor and Support Vector Classification utilizing scikit-learn. Scikit-learn is a great tool written by very intelligent people. If you don't know what sklearn is I strongly suggest you go check it out.


Nearest Neighbor or KNN

In this case Nearest Neighbor is an unsupervised machine learning algorithm used to predict over your dataset.


Support Vector Classifcation or SVC

In this case Support Vector Classification is a supervised machine learning algorithm used to predict over your dataset.
